#### 1번 문제
1번문제는 고속도로 통과 문제였다
```
30 50
40 60
70 120
```
과 같은 최소속도 최고속도가 정해져있고
```
60
70
40
```
와 같은 각 자동차별 속도가 주어질 때 
각 자동차가 통과하는 속도기준의 개수를 모두 더하는 문제였다

나는 모든것을 탐색해야했고 검색해야하는 데이터 개수도 많지 않았기 떄문에
Brute-Force라고 생각을 해서 이중 for문 으로 모든것을 다 검사해 풀었지만
O(N^2)이기 때문에 불길한 예감은 지울 수 없었다
